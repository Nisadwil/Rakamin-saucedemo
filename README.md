**Automation Testing "Rakamin-saucedemo"
**
Automation Testing menggunakan Selenium dan Cucumber pada website https://www.saucedemo.com/ sebagai salah satu latihan pembelajaran dari Rakamin Academy.

Terdapat 5 Features yang di automasikan (belum semua feature):
1. Login User (3 Scenarios)
2. Filter Product (2 Scenarios)
3. Show Product (2 Scenarios)
4. Add Product to Cart (3 Scenarios)
5. Remove Product from Cart (3 Scenarios)

Tools yang perlu diinstal terlebih dahulu:
1. Java (JDK)
2. IDE Java
3. Maven

Cara Run:
1. Import semua Code pada IDE Java
2. Klik kanan pada Folder Rakamin-saucedemo kemudian pilih Run All Tests
